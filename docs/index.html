ğŸ“š Usage Examples

Get started with Laravel Spectrum in minutes. No configuration required - just install and generate.

1

Basic Documentation Generation

Generate comprehensive API documentation from your existing Laravel routes and controllers with a single command. 

# Install Laravel Spectrum
composer require laravel-spectrum/spectrum --dev

# Generate OpenAPI documentation
php artisan spectrum:generate

# View your documentation
# Documentation available at /docs
2

Export to External Tools

Export your generated documentation to popular API testing tools like Postman and Insomnia for seamless integration. 

# Export to Postman collection
php artisan spectrum:export --format=postman --output=api-collection.json

# Export to Insomnia workspace
php artisan spectrum:export --format=insomnia --output=workspace.json

# Export OpenAPI 3.0 specification
php artisan spectrum:export --format=openapi
3

Mock Server for Development

Start a built-in mock server based on your API documentation for frontend development and testing. 

# Start mock server on default port
php artisan spectrum:mock

# Start mock server on custom port with watch mode
php artisan spectrum:mock --port=8080 --watch

# Mock server running at http://localhost:8080
# Automatically updates when code changes are detected
