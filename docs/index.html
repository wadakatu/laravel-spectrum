Usage Examples

Quick Start

Install and generate your first API documentation with zero configuration.

# Install via Composer
composer require --dev laravel-spectrum/spectrum

# Generate OpenAPI documentation
php artisan spectrum:generate

# View your documentation
http://your-app.test/docs
ğŸ”§

Custom Configuration

Customize your documentation output with optional configuration settings.

# Publish configuration file
php artisan vendor:publish --tag=spectrum-config

# Generate with custom settings
php artisan spectrum:generate \
  --output=./docs/api.json \
  --title="My API Documentation" \
  --version=1.0.0
ğŸ“¤

Export & Mock Server

Export your API documentation to popular tools and start a mock server for testing.

# Export to Postman collection
php artisan spectrum:export --format=postman

# Export to Insomnia collection
php artisan spectrum:export --format=insomnia

# Start mock server
php artisan spectrum:mock --port=8080
