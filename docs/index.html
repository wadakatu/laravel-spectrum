Usage Examples

Get started with laravel-spectrum in minutes with these simple examples

üöÄ Basic Setup & Generation

Install and generate your first API documentation

# Install via Composer
composer require --dev laravel-spectrum/spectrum

# Publish configuration (optional)
php artisan vendor:publish --provider="Spectrum\SpectrumServiceProvider"

# Generate documentation
php artisan spectrum:generate

‚öôÔ∏è Configuration Options

Customize your documentation generation settings

// config/spectrum.php
return [
    'title' => 'My API Documentation',
    'version' => '1.0.0',
    'routes' => [
        'prefix' => 'api',
        'exclude' => ['admin/*', 'internal/*'],
    ],
    'output' => [
        'path' => 'public/docs',
        'format' => ['json', 'yaml', 'html'],
    ],
];

üîÑ Real-time Updates & Mock Server

Enable live documentation updates and start the mock server

# Watch for changes and auto-regenerate docs
php artisan spectrum:watch

# Start built-in mock server
php artisan spectrum:serve --port=8080

# Export to external tools
php artisan spectrum:export --format=postman --output=api-collection.json
php artisan spectrum:export --format=insomnia --output=api-workspace.json
