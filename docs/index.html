Usage Examples

Get started with Laravel Spectrum in minutes. Here are common usage patterns to generate your API documentation.

1

Basic Setup & Generation

Install and generate your first API documentation with zero configuration required.

# Install via Composer
composer require --dev laravel-spectrum/spectrum

# Generate documentation
php artisan spectrum:generate

# Serve with built-in mock server
php artisan spectrum:serve
2

Export to External Tools

Export your generated documentation to popular API testing tools like Postman and Insomnia.

# Export to Postman collection
php artisan spectrum:export --format=postman --output=api-collection.json

# Export to Insomnia workspace
php artisan spectrum:export --format=insomnia --output=workspace.json

# Generate OpenAPI 3.0 spec
php artisan spectrum:export --format=openapi --output=openapi.yaml
3

Watch Mode for Development

Enable real-time documentation updates during development with automatic file watching.

# Watch for changes and auto-regenerate
php artisan spectrum:watch

# Watch specific routes or controllers
php artisan spectrum:watch --paths=app/Http/Controllers/Api

# Combine watch with serve for live development
php artisan spectrum:watch --serve --port=8080
